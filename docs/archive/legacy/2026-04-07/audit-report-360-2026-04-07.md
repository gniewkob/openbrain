| Secret Scanning | ✅ | Middleware wykrywające sekrety |

### 2.2 Kluczowe Zabezpieczenia

```python
# Timing-safe comparison
hmac.compare_digest(provided_key, expected_key)

# Secret scanning patterns
- OpenAI API keys (sk-...)
- GitHub tokens (ghp_...)
- JWT tokens (eyJ...)
- PEM private keys
- URL credentials

# Public mode - fail-closed
- Wymaga INTERNAL_API_KEY (min 32 znaki)
- Zakaz domyślnych credentiali
- Wymaga OIDC_ISSUER_URL lub INTERNAL_API_KEY
```

### 2.3 Autentykacja Wielowarstwowa

1. **Internal API Key** - `X-Internal-Key` header
2. **OIDC/JWT** - Auth0 integration
3. **OAuth 2.1** - dla ChatGPT
4. **Rate Limiting** - per-IP sliding window
